#Starting from Visual Studio Code

This assumes Visual Studio Code is already open and you are inside the dev folder we set up on the first day of class. Additionally, these instructions will work with another code editor or an HTML template from the web, it is just that the default CSS and JS files will have different names.
  1. Navigate to the folder you want to put your homework in.
    • Labs for labs
    • Exercises for Exercises
    • Code Alongs for Code alongs
    • Activities for Activities (if you set one up)
  2. Right click on that folder and select Add Folder"
  3. Enter the name of the project
    • e.g. "DOM-Exercise-01", "jQuery-Lab-Part-01", etc.
  4. Press Enter
  5. Once the folder has been created, right click on it and select New File
  6. Type index.html then press Enter
  7. Go into the index.html file you just created
  8. Type the letters "html" and you will see a menu pop up right where you are typing. Choose second option with a square that has a dotted bottom (representing a code block) and it will insert an HTML template into the file for you
    • Either navigate to it using your arrow keys and pressing enter OR
    • Click it with your mouse
  9. If you will not be using CSS or JavaScript
    1. Comment out or remove completely the <script> tag that loads main.js
    2. Comment out or remove completely the <link> tag that loads main.css
  10. If you will be using Javascript...
    1. Go into the index.html file and move the script tag that loads main.js line down to just before the closing </body> tag
    2. Right click the top folder of your project and select New File again
    3. Type main.js and hit Enter
  11. If you will be using CSS...
    1. Right click the top folder of your project and select New File again
    2. Type main.css and hit Enter
  12. Now, right click on the top folder of your project and select "Open in Terminal"
  13. Within the terminal, type git init and hit enter
  14. Open up a web browser and head to GitHub
  15. Create a new repository there that has the same name as your project's top folder
  16. After it has been created, copy the link under Quick setup — if you’ve done this kind of thing before
    • Note that you can just click the button at the end of the URL to copy it, you don't have to highlight it!
  17. Back in Visual Studio code, type git remote add origin then paste the URL. Hit enter
    • Make sure to include the space after origin and before the URL!
  18. Next, type git status and hit enter to ensure your project is ready for git
  19. If you've been following these steps, the file(s) you've added should be in red. That's because they need to be added to git to be tracked
  20. Do this by typing git add . and hitting enter
    • The period means "all"
  21. Type git status again, and your file(s) should now be in green
  22. Next, type git commit -a --m "init commit" and hit enter
    • The -a means all
    • The --m means you're going to add a message
    • The " is the start and end of your commit message
    • Your message should change after this, where you say what you've actually changed since the last time you commited
  23. Type git status
